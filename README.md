# Veterinary Practice Client Communication Bot

Telegram bot for veterinary clinics automating pet owner communication and appointment management. Sends appointment reminders with pre-visit preparation instructions, post-visit care guidance with medication schedule, annual vaccination due alerts, and heartworm and flea treatment reminders. Pet owners submit urgent symptom queries and get triage guidance with emergency contact information. Clinic staff receive daily booking digest and no-show alerts. Supports multi-pet households with individual profiles per animal. Fully configurable practice profile with vet name, hours, and emergency contacts.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
